Reemplazar y Actualizar Filas en MySQL

Reemplazar Filas

En MySQL, reemplazar filas se refiere a la acción de insertar nuevos registros en una tabla o actualizar registros existentes si ya hay una coincidencia basada en una clave única.

Ejemplo de Reemplazar Filas

Supongamos que tienes una tabla llamada clientes con las columnas id, nombre y email, donde id es una clave única. Si deseas agregar un nuevo cliente o reemplazar uno existente con el ID 1, puedes usar la siguiente consulta:

REPLACE INTO clientes (id, nombre, email)
VALUES (1, 'Juan Pérez', 'juan@example.com');

Actualizar Filas

La actualización de filas en MySQL se utiliza para cambiar los valores de una o varias columnas en una o varias filas de una tabla.

Ejemplo de Actualizar Filas

Supongamos que tienes una tabla llamada empleados con las columnas id, nombre, salario y departamento. Si deseas aumentar el salario de todos los empleados en el departamento de "Ventas" en un 10%, puedes hacerlo con la siguiente consulta:

UPDATE empleados
SET salario = salario * 1.1
WHERE departamento = 'Ventas';
