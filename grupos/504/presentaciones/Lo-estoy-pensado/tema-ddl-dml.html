ALTER TABLE - RENAME

Por ejemplo, si queremos cambiar el nombre de la tabla: mysql>ALTER TABLE (nombre) RENAME (nuevo nombre);

ALTER TABLE - DROP COLUMN

Para eliminar una columna de una tabla: mysql>ALTER TABLE(nombre de la tabla) DROP COLUMN (nombre de la columna);

volver

ALTER TABLE - DROP PRIMARY KEY

Para eliminar la clave primaria de una columna: mysql>ALTER TABLE (clave primaria) DROP PRIMARY KEY;

volver

ALTER TABLE - ADD

Para insertar una nueva columna al final de una tabla: mysql>ALTER TABLE (nombre de la tabla) ADD (nueva columna); .

ALTER TABLE - ADD - AFTER

Para añadir una nueva columna después de otra: mysql>ALTER TABLE (nombre de la tabla) ADD (columna) (tipo de dato) AFTER (columna nueva);

volver

ALTER TABLE - ADD - FIRST

Para añadir una nueva columna en la primera posición de la tabla: mysql>ALTER TABLE (nombre de la tabla) ADD (columna) (tipo de dato) FIRST;

volver

ALTER TABLE - ADD - PRIMARY KEY

Asignar como clave primaria a una columna: mysql>ALTER TABLE (nombre de la tabla) ADD PRIMARY KEY(nombre);

volver

La forma más directa de insertar una fila nueva en una tabla es mediante una sentencia INSERT. En la forma más simple de esta sentencia debemos indicar la tabla a la que queremos añadir filas, y los valores de cada columna.

EJEMPLO 1

mysql>INSERT INTO (nombre de la tabla) VALUES ((atributo), (atributo), (atributo));

Introducir más de una fila a la vez:

mysql>INSERT INTO (nombre de la tabla) VALUES → ((atributos)), →(atributos);

Configure dominios de datos para agrupar columnas de origen de datos para el enmascaramiento de datos.

regresar

También es posible especificar la lista de columnas/atributos donde deseamos ingresar ciertos valores.

mysql>INSERT INTO granciudad (nombre de la(s) columnas) VALUES (atributos);

Además existe una alternativa que consiste en indicar un valor para cada columna:

mysql>INSERT INTO (nombre de la tabla) SET
(columna) = (atributo), (columna) = (atributo);
